THE MIRAL CLEANING - EXTREME WEBSITE

Deze website gebruikt de huisstijl van je logo:
- roze bloemkleuren
- zwart/wit professionele stijl
- sierlijk titellettertype en modern tekstlettertype

Bestanden:
- index.html
- style.css
- script.js
- assets/logo.png

Zo gebruik je de website:
1. Pak dit zip-bestand uit.
2. Open index.html in je browser.
3. Navigatie, mobiel menu, animaties, terug-naar-boven knop en offerteformulierbevestiging werken.

Let op:
Het formulier toont lokaal een bevestiging, maar verstuurt nog geen echte mail. Daarvoor heb je hosting/Formspree/Netlify Forms of een backend nodig.
